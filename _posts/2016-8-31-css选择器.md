## 子代选择器只能选中直接后代元素 ##

		/* 后代选择器，包含直接和非直接子 */
		/* .one span{
			color: red;
		} */
		
		/* 子代选择器只能选中直接后代元素 */
		/* .one>span{
			color: green;
		} */
		div>span{
			color: green;
		}
	</style>
